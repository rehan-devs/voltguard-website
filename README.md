## About

This is a single-page, fully responsive website built for a premium electrical subcontractor. It's designed to feel trustworthy, warm, and human — like a neighborhood shop that's been around forever meets the quiet confidence of the guy on the crew everyone trusts with the panel.

No dark mode. No futurism. No glowing neon. Just clean work.

---

## Features

- **Single HTML file** — no build tools, no frameworks, no dependencies
- **Fully responsive** — mobile, tablet, and desktop
- **Accessible** — WCAG AA compliant, keyboard navigable, skip-to-content link
- **Lightweight** — under 2MB total page weight target
- **Zero JavaScript dependencies** — vanilla JS only (Intersection Observer for scroll reveals and counter animation)
- **Three font families** — Libre Baskerville, DM Sans, IBM Plex Mono via Google Fonts
- **Reduced motion support** — respects `prefers-reduced-motion`
